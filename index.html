Games Shelves

Games Shelves helps you effortlessly organize and search your board game collection. Find games based on name, location, or category and streamline your inventory:

  • Quickly locate games by scanning or typing barcodes
  • Switch between compact and detailed views for additional information
  • Hover over columns for quick tooltips with helpful information

  • Python and Flask backend powers the web application
Meet in the Middle

Meet in the Middle helps you find the optimal meeting point between multiple people based on starting locations.

  • Supports up to 10 starting points
  • Displays a map with the starting points and calculated meeting location
  • Shows points of interest near the meeting point within an adjustable range

Highlights:

  • Developed with JavaScript, CSS, and HTML
  • Integrates Google Places API for points of interest
  • Backend powered by Python and Flask
